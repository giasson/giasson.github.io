Before/After Demo Task Force (BAD TF) Work Statement

Status

This Work Statement for the Before/After Demo Task Force (BAD TF) (hereafter referred to as "Task Force") has been reviewed and approved by EOWG in its teleconference on 1 July 2005. This Task Force is closed since January 2012; the latest version of the work is available at http://www.w3.org/WAI/demos/bad/.

Objectives

The Task Force operates under the EOWG charter.

The objective of the Task Force is to:

  • develop an EOWG Resource Suite demonstrating a "before" and "after" repair of a Web site for the purpose of increasing accessibility, and explaining the changes in the site, as described in the Requirements for Before/After Demo document.

Communication

The Task Force communications are publicly visible. Communication mechanisms for the Task Force include:

  1. The Task Force mailing list<public-wai-eo-badtf@w3.org> and archives
  2. Weekly (or biweekly) Task Force teleconferences
  3. Monthly (or more frequently, as needed) Task Force updates and feedback in EOWG teleconferences
  4. BADTF Web page

Participation

Task Force participants must be participants of EOWG; should be skilled in HTML, CSS, scripting, Web development, and WCAG; and must actively contribute to the work of the Task Force, including:

  1. Minimum 4 hours per week of work on Before/After Demo (Task Force is intended to be short-term (approximately 3 months))
  2. Remain current on Task Force  mailing list and respond in a timely manner to postings on Task Force mailing list
  3. Participate in Task Force telephone meetings, or send regrets to the Task Force list
  4. Assist in preparing Task Force deliverables for discussion

W3C Team Leadership

The Task Force facilitator is Shadi Abou-Zahra.
